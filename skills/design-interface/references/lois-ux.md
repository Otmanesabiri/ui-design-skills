# UX Laws (Applied Psychology)

**Agent Instruction:** Use for UX strategy. For component specs, see `composants.md`.

---

## 1. Interaction Models
`Law[Jakob]: Habit > Innovation. Rule: Respect mental models (Standard Nav, Top-Left Logo, X-Close).`
`Law[Hick]: Choices ∝ decision_time. Rule: Progressive_Disclosure. Chunk(Max_5_Options).`
`Law[Tesler]: Complexity = Incompressible. Rule: System absorbs complexity (e.g. Autocomplete) to save User.`

## 2. Target Acquisition & Physicality
`Law[Fitts]: Target(Size, Distance) ∝ time. Rule: Min_Target(44x44px). Primary_Action = Bottom/Thumb_Zone.`
`Law[Doherty]: System_Response < 400ms. Rule: Perceived_Speed. Latency_UI(Skeleton, Optimistic_Updates).`

## 3. Working Memory & Perception
`Law[Miller]: Chunking(Max_5-7_items). Rule: Scannable blocks (e.g. Phone: 00 00 00 vs 000000).`
`Law[VonRestorff]: Distinct_Object = Most_Remembered. Rule: Reserve highlight for 1-2 elements/view.`
`Law[Postel]: Produce_Strictly. Accept_Flexibly. Rule: Input_Flexibility (Ignore phone spaces/dashes).`

## 4. Emotional Experience
`Law[Peak-End]: Judgment = Most_Intense + Final_Moment. Rule: Celebratory Success_Flow. Actionable_Errors.`
`Law[Aesthetic-Usability]: Beauty = Perceived_Ease. Rule: High-polish reduces user friction/bug-frustration.`

## Gestalt Principles (Visual Logic)
| Principle | UI Logic |
|---|---|
| Proximity | Close = Related (Grouped inputs) |
| Similarity | Same Style = Same Function |
| Continuity | Eye follows line (Scroll rails, alignment) |
| Common Region | Containers = Context (Cards) |
| Figure-Ground | Elevation = Hierarchy (Modals, Overlays) |

## Ethical Framework (Anti-Patterns)
`Rule: Subscribing_Ease == Unsubscribing_Ease. Forbid(Manipulation).`

| Pattern | Description | Logic |
|---|---|---|
| Confirmshaming | Shaming refusal ("No, I hate saving") | ❌ Dark |
| Hidden Costs | Fees revealed at final step | ❌ Dark |
| Roach Motel | Easy in, hard out (laborious cancel) | ❌ Dark |
| Misdirection | Drawing eye from real choices | ❌ Dark |
| Trick questions | Misleading pre-checks | ❌ Dark |
